# TV_Server

Медіасервер для локальної мережі, оптимізований для Lampa (Smart TV / PC). 

## Склад
Цей стек розгорнуто за допомогою Docker Compose і містить:
- **TorrServer** (порт `8090`) - для потокового відтворення торентів без завантаження на диск. Налаштований кеш: 2ГБ.
- **Jackett** (порт `9117`) - для уніфікованого пошуку торентів (додано Toloka.to, ThePirateBay, 1337x).

## Як запустити
1. Встановіть Docker та Docker Compose.
2. Створіть файл `media-server/.env` за прикладом `media-server/.env.example` та вкажіть там потрібні дані (якщо використовуєте автоматизацію).
3. Перейдіть в папку `media-server` і виконайте команду:
   ```bash
   docker compose up -d
   ```
4. Відкрийте Jackett (`http://localhost:9117`) та налаштуйте потрібні трекери, або TorrServer (`http://localhost:8090`) для перевірки статусу.

## Підключення до Lampa
- **TorrServer IP:** Вкажіть IP-адресу вашого ПК (наприклад, `192.168.1.165`) та порт `8090` в налаштуваннях Lampa.
- **Jackett:** Вкажіть IP-адресу ПК, порт `9117` та API-ключ Jackett у плагіні для пошуку торентів.
